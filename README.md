# Physical connections

- Digital pin connected to the input signal: pin 2 (can be changed in the code of course)

- SD card attached to SPI bus as follows:
  - MOSI - pin 11
  - MISO - pin 12
  - CLK - pin 13
  - CS - pin 4

- I2C SSD1306 Oled DISPLAY:
  - SCL = A5
  - SDA = A4

- I2C RTC module
  - SCL = A5
  - SDA = A4
